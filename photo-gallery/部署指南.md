# 画廊部署指南

完成后你会得到：
- **公开画廊**：`https://你的名字.vercel.app`（访客访问）
- **管理后台**：`https://你的名字.vercel.app/admin`（只有你能用）

预计时间：**30 分钟**，不需要写代码。

---

## 第一步：注册 Supabase（数据库）

1. 打开 [https://supabase.com](https://supabase.com)，点击 **Start your project**，用 GitHub 或邮箱注册
2. 点击 **New project**，填写：
   - Project name：随便写，如 `my-gallery`
   - Database Password：设一个强密码（记下来备用）
   - Region：选 **Northeast Asia (Tokyo)**（离你最近）
3. 等待约 1 分钟，项目创建完成

4. 进入项目后，点击左侧菜单 **SQL Editor**
5. 把 `supabase_schema.sql` 文件里的全部内容**复制粘贴**进去，点击 **Run**
   - 出现绿色 "Success" 即成功

6. 记录你的密钥：
   - 左侧菜单 → **Settings** → **API**
   - 复制 **Project URL**（形如 `https://xxxx.supabase.co`）
   - 复制 **anon public** key（很长的一串字符）

---

## 第二步：注册 Cloudinary（图片存储）

1. 打开 [https://cloudinary.com](https://cloudinary.com)，注册免费账号
2. 进入控制台，左上角可以看到你的 **Cloud name**（记下来）
3. 创建上传预设：
   - 左侧菜单 → **Settings** → **Upload**
   - 滚动到 **Upload presets**，点击 **Add upload preset**
   - Signing Mode 选 **Unsigned**
   - Preset name 写 `gallery_unsigned`
   - 点击 **Save**

---

## 第三步：部署到 Vercel

1. 打开 [https://vercel.com](https://vercel.com)，用 GitHub 或邮箱注册

2. 你不需要 GitHub！直接用 Vercel CLI 上传本地文件夹：

   **方法 A：拖拽上传（最简单）**
   - 进入 Vercel 控制台，点击 **Add New → Project**
   - 页面下方有 **Deploy from your local project** 或直接将 `photo-gallery` 文件夹**拖拽**到页面
   - 点击 Deploy，等待约 30 秒

   **方法 B：用命令行（如果拖拽不成功）**
   ```bash
   # 安装 Vercel CLI
   pip install nodeenv
   nodeenv venv_node
   source venv_node/bin/activate
   npm install -g vercel

   # 进入项目目录部署
   cd photo-gallery
   vercel
   # 按提示操作，一路回车即可
   ```

3. 部署完成后，Vercel 会给你一个网址，如 `https://photo-gallery-xxx.vercel.app`

---

## 第四步：填写配置

### 4a. 配置公开画廊（让访客能看到图片）

打开 `public/config.js`，替换为你的实际值：

```js
window.GALLERY_CONFIG = {
  supabaseUrl: 'https://你的project_id.supabase.co',
  supabaseKey: '你的anon_public_key',
};
```

保存后重新上传到 Vercel（再次拖拽或 `vercel --prod`）。

### 4b. 配置管理后台

访问 `https://你的域名.vercel.app/admin`

1. 默认密码是 `admin123`，进入后立刻去 **配置** 页修改
2. 在配置页填写：
   - Supabase URL
   - Supabase Anon Key
   - Cloudinary Cloud Name
   - Cloudinary Upload Preset（`gallery_unsigned`）
   - 新的管理密码
3. 点击 **保存配置**，再点击 **测试连接** 确认连通

---

## 第五步：开始使用

1. 访问 `/admin` → **上传图片**
2. 拖拽图片 → 填写日期和标签 → 上传
3. 访问你的主网址，就能看到公开画廊了 🎉

---

## 常见问题

**Q：图片上传失败？**
检查 Cloudinary Cloud Name 和 Upload Preset 是否正确，Preset 必须是 Unsigned 模式。

**Q：画廊打开是空的？**
检查 `public/config.js` 里的 Supabase URL 和 Key 是否填正确，注意不要有多余空格。

**Q：管理后台密码忘了？**
打开浏览器开发者工具 → Console，输入：
`localStorage.setItem('gallery_admin_config', JSON.stringify({adminPwd:'新密码'}))` 然后刷新页面。

**Q：Supabase 免费额度够用吗？**
免费版：数据库 500MB（元数据绰绰有余）、存储 1GB（我们不用 Supabase 存图，用 Cloudinary）。
Cloudinary 免费版：25GB 存储，1000 张高清图片约占 5GB，完全够用。

**Q：如何更新网站？**
修改 HTML 文件后，重新拖拽上传到 Vercel，或运行 `vercel --prod`。

## 文件结构说明

```
photo-gallery/
├── public/
│   ├── index.html      ← 公开画廊（访客看到的）
│   └── config.js       ← 填写你的 Supabase 密钥
├── admin/
│   └── index.html      ← 管理后台（只有你用）
├── vercel.json         ← Vercel 路由配置
└── supabase_schema.sql ← 数据库建表 SQL
```
